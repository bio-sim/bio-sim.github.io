Descriere

Este un săpun extrem de hidratant, face o spumă cremoasă ce conferă o senzație plăcută pe piele. Este un săpun cu proprietăți emoliente, calmante și cicatrizante asupra pielii.

Uleiul de ricin este un emoliant natural, stimulează producția de colagen și elastină, menține umiditatea pielii. Înmoaie pielea, ceea ce ușurează exfolierea celulelor moarte. Menține umiditatea pielii prin proprietatea de a fi un bun umectant.

Untul de cacao este supranumit „hidratantul extrem”. Are o putere mare de hidratare, fiind și puternic antioxidant prin tacoferolii constituienți.

Argila roșie este o argilă delicată, cu proprietăți tonifiante, bogată în minerale și oligoelemente. Bogată în oxizi de Fe, are rol de iluminare a pielii, conferindu-i catifelare. Este recomandată pentru pielea cu tendințe de uscare, sensibilă, ameliorează circulație sângelui, calmează inflamațiile și iritațiile.

Laptele de capră este supranumit „elixirul alb”, aduce multe beneficii săpunului prin conținutul de nutrienți, prin calitățile deosebite de hidratare. Este aniacneic, calmează iritațiile. Beneficiile laptelui de capră le vor simți cel mai bine persoanele care suferă de diferite afecțiuni ale pielii. Laptele de capră este similar ca și compoziție cu laptele matern, de aceea este eficient în protejarea epidermei împotriva bacteriilor și a substanțelor toxice de orice natură.

Un săpun cu lapte de capră este mult mai hidratant decât orice alt săpun.
