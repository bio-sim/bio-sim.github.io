Descriere

Este prin excelență un săpun cu putere mare de detoxifiere (datorită cantității mari de argilă albastră), antiinfecțios, antiinflamator.

Uleiurile conținute în acest săpun hrănesc, înmoaie și protejează pielea.

Uleiul de ricin este antiinflamator, detoxifiant, vindecă micile leziuni ale pielii. Este emolient.

Untul de shea protejează pielea, este nutritiv, emolient, regenerează țesuturile pielii deteriorate.

Uleiul de chimen negru este antiinflamator, antifungic, purifiant. Ajută pielea cu probleme, în special cea cu tendințe acneice. Stimulează regenerarea țesuturilor. Este anti-age, restructurant cutanat, îmbunătățește condiția pielii în caz de dermatite, eczeme, psoriazis.

Argila albastră de Râciu este cea mai concentrată în minerale și oligoelemente (41), oxizi, silicați. Este cea mai pură și cea mai eficientă argilă din lume! Dintre minerale: Ca, Mg, K, Fe, Si, S, Ti, Na, Cl, Al. Dintre oligoelemente: Mn, Cu, I, Se, Cr, Mb, P, Ge, Zn, Ag, Br, V, Ru, B.

Argila albastră are următoare efecte: antiseptice, antibacteriene, antimicotice, antiinflamatoare. Are o putere deosebită în detoxifierea corpului de metale grele, radicali liberi. Poate neutraliza substanțe toxice ingerate, mărește imunitaea sistemică și de țesut. Are calități cosmetice deosebite. Dezinfectează și detoxifică pielea prin îndepărtarea bacteriilor și toxinelor, având rezultate benefice în afecțiuni ca: furunculi, herpes, psoriazis, eczeme, acnee vulgară, rozacee, piele uscată, prurit, infecții micotice, dermatite, dermatoze, cuperoze.

Uleiurile esențiale utilizate în parfumarea săpunului aduc o notă plăcută, un mix de arome ce acționează sinergic pentru un efect detoxifiant puternic.

Uleiul esențial de oregano este antiseptic, antiinfecțios, fungicid, paraziticid, tonifiant.

Uleiul esențial de rozmarin este purifiant, antiseptic, bactericid, fungicid, tonic.

Uleiul esențial de lime este antiseptic, bactericid, dezobturant al porilor, antioxidant, îmbunătățește circulația, este antiinflamator.

Uleiul esențial de lavandă este echilibrant, regenerator și reparator cutanat, calmant, cicatrizant.

Uleiul esențial de lemongrass are o aromă citrică ierboasă, este antiseptic, tonic, dezodorizant, stimulează microcirculația, este echilibrant al sistemului nervos.

Uleiul esențial de niaouli este antiviral, bactericid, fortifică sistemul imunitar, este util în acnee, plăgi, furunculi, eczeme, este purifiant tonic.
