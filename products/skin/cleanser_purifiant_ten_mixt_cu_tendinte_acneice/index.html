Cleanser purifiant - ten mixt cu tendințe acneice

Ingrediente
  • apă de hamamelis (hidrolat)
  • apă de rozmarin (hidrolat)
  • xanthan
  • tensidă
  • acid salicilic BHA
  • extracte din alge marine
  • ulei esențial de tea tree
  • ulei esențial de lime (lămâie verde)
  • conservant

Descriere

Un cleanser-gel potrivit pentru curățarea tenului, indicat în special pentru tenul mixt/gras, cu puncte negre.

Igienizează pielea, este purifiant și seboreglator, exfoliază ușor enzimatic.

Apa de hamamelis are efect tonic, astringent, favorizează microcirculația, este benefică tenului mic/gras.

Apa de rozmarin este indicată pentru curățarea și îngrijirea tenului mixt/gras sau cu imperfecțiuni. Echilibrează secreția de sebum, purifică și este astringentă.

Acidul salicilic BHA contribuie la regenerarea celulară, îmbunătățește aspectul pielii. Este antiacneic și antimicrobian forte, antiinflamator, exfoliant, desfundă porii prin îndepărtarea celulelor moarte și a altor depuneri ce îi blochează.

Extractul de alge marine are proprietăți antibacteriene și antiinflamatoare. Este util în combaterea acneei, echilibrând secreția de sebum, îmbunătățind condiția și aspectul tenului mix/gras, calmând iritațiile. Este bine tolerat și de tenul sensibil sau alergic.

Uleiul esențial de tea tree este purifiant, antiseptic, util în acnee, infecții, fiind un antibacterian cu spectru larg. Este antifungic și antiviral.

Uleiul esențial de lime este benefic tenului mixt/gras, este antiseptic, bactericid, dezopturează porii, stimulează microcirculația, este antiinflamator.

Mod de utilizare

Se folosește o cantitate mică, cât un bob de mazăre, care se întinde în palmele umede, apoi se masează tenului cu mișcării blânde, circulare pentru o spumare fină uniformă. Se clătește cu apă călduță.
